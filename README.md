# Hi, I'm Kidoarch 👋

### 🎓 Educator · Simulation Builder · EdTech Explorer

I build **interactive math and science simulations** that make abstract concepts tangible for students. From visualizing calculus to simulating physics experiments — I believe the best way to understand something is to *play with it*.

---

## 🔭 What I Build

| Area | Examples |
|------|---------|
| 📐 **Mathematics** | Geometry explorers, function plotters, calculus visualizers |
| ⚗️ **Physics** | Motion simulators, wave interference, optics ray tracers |
| 🧪 **Chemistry** | Molecule builders, reaction simulations |
| 🌍 **Earth Science** | Climate models, tectonic visualizations |
